# SQL-Portfolio-Grundlagen
SQL Grundlagen – Analyse von Mitarbeiter- und Gehaltsdaten

Dieses Projekt zeigt meine ersten praktischen Schritte in der Datenanalyse mit SQL.

Im Fokus steht die Aufbereitung einer Mitarbeiterdatenbank sowie die Analyse von Gehaltsstrukturen und Karriereverläufen. Dabei habe ich grundlegende SQL-Techniken wie Joins, Aggregationen und erste Window Functions eingesetzt.

Ziel war es, eine saubere Datenbasis zu schaffen und erste aussagekräftige Kennzahlen für Management-Entscheidungen zu entwickeln.
